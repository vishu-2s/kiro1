5. Usage Notes
Automatic Application:
  • Spider dividers are automatically added between Dashboard sections (after .control-panel)
  • Web dividers are automatically added between Report sections (.report-section + .report-section)
  • Web connectors are automatically added between recent scan items and finding cards
  • Glitched symbols are applied to loading states and status indicators
Manual Usage:
  • Add <div class="spider-divider"></div> for a standard divider
  • Add <div class="web-connector-horizontal"></div> for horizontal connectors
  • Add <div class="glitch-symbol" data-text="游동">游동</div> for glitched symbols
Accessibility:
  • All decorative elements have pointer-events: none
  • Animations are disabled when prefers-reduced-motion is set
  • Elements are hidden in high contrast mode
Performance:
  • All animations use GPU acceleration with transform: translateZ(0)
  • Backface visibility is hidden to prevent flickering
  • Will-change property is used for animated elements
